JNguyen Co. CRM

System Manual — Wedding & Event Photography / Videography, Canberra

Purpose: This manual covers every feature in the CRM — from adding a client to delivering the final gallery. Read it top-to-bottom once, then use the sidebar to jump back to any section.

Overview

The CRM manages the complete lifecycle of every booking:

→
Quote
→
Contract
→

→
Event Day
→

Each stage has dedicated buttons inside the Booking Detail page so nothing gets missed.

Login & Access

1

Open the app

Navigate to http://localhost:3000 (development) or your deployed URL in a browser.

2

Sign in with your Supabase account

Enter your email and password. All data is scoped to your account — nobody else can see your clients or bookings.

3

Connect Google (first time only)

Go to Settings → Integrations and connect your Google account. This enables the Gmail API for sending emails from your Gmail address.

Clients Module

Every booking must be linked to a client. Create the client first, then create the booking.

Add a New Client

1

Go to Clients → New Client

Click + New Client in the sidebar or the Clients index page.

2

Fill in the details

Required: first name, last name, email. Optional: phone, address, partner name, event type.

3

06 — Quote & Deposit section

Select a package to auto-fill the Quoted Total. Adjust the deposit amount if needed (default is 30%). These pre-fill the linked booking.

4

Save

You'll be redirected to the client detail page. From there, use the + New Booking button to create a booking for this client.

Tip: If the client fills out your enquiry form, create their client record immediately — even before you've quoted them. It keeps your pipeline clean.

Bookings Module

A booking tracks everything about a job: dates, venue, package, pricing, deliverables, and emails sent.

Create a New Booking

1

Bookings → New Booking (or from Client page)

If you open New Booking from a client's page, the client field is pre-filled.

2

01 — Event Details

Set event type (Wedding / Event / Portrait), event date, start time, and number of hours booked.

3

02 — Venue

Enter venue name and address. This appears on quotes and confirmation emails.

4

03 — Package

Select the package. The package name drives deliverable auto-population later.

5

06 — Pricing

Enter the Quoted Total (AUD) and Deposit Amount (AUD). These print on the PDF quote.

6

Save

Booking is created with status INQUIRY. You'll land on the Booking Detail page.

Booking Actions — Step by Step

1 · Send Quote

1

Click "Send Quote" on the Booking Detail page

Located top-right in the BookingActions panel (Stage 1).

2

What happens automatically

The system generates a branded PDF quote using the booking data, emails it via GoDaddy SMTP (johnny.nguyen@jnguyen.co), and updates the booking status to QUOTED.

3

The PDF includes

Quote number, event details, package name, deliverables list, total price, deposit amount, balance, and bank transfer details.

2 · Send Confirmation

Once the contract is signed and deposit received, click Send Confirmation. This sends a booking-confirmed email to the client and sets status to CONFIRMED.

3 · Pre-Event Checklist

Send this 1–2 weeks before the event. The email asks the client to confirm:

  • Run sheet / timeline
  • Vendor contacts
  • Venue access details
  • Any last-minute timeline changes

4 · Setup Deliverables

Click Setup Deliverables to auto-create deliverable records based on the booking's package. Due dates are calculated from the event date.

PackageDeliverables Created
Mini Wedding / Elopement Teaser (+2 days), Photo Gallery (+28 days), Highlight Film (+42 days)
Full Day Essential (8 hr) Teaser (+2 days), Photo Gallery (+42 days), Highlight Film (+56 days)
Full Day Premium (13 hr) Teaser (+2 days), Gallery (+42 days), Film (+56 days), Raw Footage (+56 days)
Hourly Photo Photo Gallery (+14 days)
Hourly Photo + Video Photo Gallery (+14 days), Highlight Film (+21 days)

Idempotent: Clicking Setup Deliverables again won't create duplicates — it only adds types that don't already exist on the booking.

5 · Send Gallery

1

Export and upload to Google Drive first

Upload the final edited files to your Google Drive and copy the share link.

2

Confirm final payment is cleared

Only send the gallery once the balance is paid.

3

Click "Send Gallery" and paste the Drive link

A modal opens for the Drive URL. The email is sent via GoDaddy SMTP with download tips and the link.

6 · Request Review

Click Request Review a few days after gallery delivery. A friendly email is sent asking the client to leave a Google review. The review link goes to: https://g.page/r/CZdsm6gVm2xqEAE/review

7 · Generate Invoice

Click Generate Invoice to open the invoice creation page pre-filled with this booking's data. Send the invoice PDF via email once generated.

Documents Module

Upload and manage contracts, run sheets, and any other files associated with a booking.

Upload a File

1

Go to Documents (sidebar)

The Documents page lists all files in your Supabase storage bucket.

2

Click Upload or drag a file in

Accepted types: PDF, images, Word documents. Files are private to your account.

Delete Files

Two deletion options are available:

  • Per-row checkbox → "Delete Selected (N)" — tick one or more rows using the checkbox on the left, then click the teal button that appears at the top.
  • "Delete All" button — deletes every file in the bucket. Use with caution.

Warning: File deletion is permanent. There is no recycle bin or undo.

Invoices Module

1

Create from a booking

Use the Generate Invoice button on the Booking Detail page to pre-fill the invoice form.

2

Review the amounts

Set the total amount, due date, and any line items. Mark what's already been paid (deposit).

3

Send Invoice PDF

Click Send Invoice. A PDF is generated and emailed to the client. Status changes to SENT.

4

Mark as Paid

When bank transfer clears, open the invoice and mark it as PAID.

Deliverables Module

Track the status of every deliverable for every booking.

StatusMeaning
NOT_STARTEDCreated but work hasn't begun
IN_PROGRESSCulling / editing underway
REVIEWReady for your internal review
DELIVEREDSent to client

Update each deliverable's status from the Booking Detail page → Deliverables section. The due dates set by Setup Deliverables show when each item is expected.
