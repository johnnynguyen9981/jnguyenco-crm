Add to Gmail — Easy

  1. Click 📋 Copy Signature above — this copies the formatted signature
  2. Open Gmail → ⚙ Settings → See all settings
  3. General tab → scroll to Signature → Create new → name it JNguyen Co.
  4. Click inside the signature text box and press Ctrl + V to paste
  5. Set as default for New emails and Replies → Save Changes

Add to Gmail — Via HTML

  1. Click Copy Raw HTML above
  2. Open Gmail → ⚙ Settings → See all settings
  3. General tab → scroll to Signature → Create new
  4. In the signature editor toolbar, look for the </> icon — if not visible, try the ⋮ More options menu
  5. Paste the HTML → click OK → Save Changes
